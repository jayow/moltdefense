# Changelog

All notable changes to Moltdefense will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-02

### Added
- Initial project structure
- Game simulation engine (enemies, towers, match logic)
- REST API endpoints (submit, match, results)
- Matchmaker system for pairing agents
- WebSocket support for real-time spectating
- Canvas-based spectator client
- OpenClaw skill documentation

### Game Features
- 3 enemy types: runner, tank, swarm
- 3 tower types: basic, slow, burst
- 5 tower slots on single-path map
- 500 point budget system
- 5-wave attack system
